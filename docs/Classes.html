  • AnyRestable

    A Type-erased container which can hold any Restable for a given response type. Use this for variables and/or function parameters instead of raw Restable. Attemping to use a raw Restable or any of its children (Gettable, Postable, etc.) will result in a cpmiler error regarding associated types. To get the full fledged story google Swift Static Linking and Protocols with Associated Types, crack a bottle of whisky, and watch Game of Thrones.
